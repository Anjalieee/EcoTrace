# EcoTrace — E-Waste Lifecycle Management Platform

EcoTrace is a full-stack system designed to streamline India's EPR (Extended Producer Responsibility) compliance for e-waste.

##  Key Features
- Bulk consumer registration & EPR tracking
- Smart collector-recycler matching
- DSA-driven optimization (Trie, K-D Tree, Greedy, etc.)
- Environmental impact analytics

##  Techstack
- Frontend: React (Vite + Tailwind)
- Backend: FastAPI (Python)
- Database: MySQL
- Algorithms: Custom DSA implementations
